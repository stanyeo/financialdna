Dark-themed, gamified financial personality quiz built with React + Vite.

## Tech Stack
- **React 18** + **Vite 6**
- **Tailwind CSS 3** (dark mode, glassmorphism, neon accents)
- **Framer Motion 11** (animations & transitions)
- **Lucide React** (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
